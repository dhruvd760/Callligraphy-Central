### Token Management Rules
*   **Targeted Reads**: Avoid reading large binaries or complete file dumps; read precise line ranges.
*   **Payload Diffs**: Communicate changes using compact diff formats rather than rewriting complete codebases.
*   **Iterative Prompts**: Build concise instructions using structured inputs to prevent token inflation.
