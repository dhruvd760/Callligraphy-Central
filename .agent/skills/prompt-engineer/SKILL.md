### Prompt Guidelines
*   **Role Allocation**: Explicitly declare model persona, target output format, and rules.
*   **Data Isolation**: Wrap input sources inside designated text delimiters (e.g., XML blocks or triple backticks).
*   **Output Typing**: Enforce structured formats (JSON, markdown) to simplify agent parser code.
