### Documentation Rules
*   **Markdown Structure**: Follow GitHub Flavored Markdown (GFM) formatting.
*   **Document Hierarchy**: Use a single H1 tag per page, followed by cascading headings.
*   **Alert Styles**: Utilize NOTE, TIP, IMPORTANT, WARNING, and CAUTION tags appropriately.
