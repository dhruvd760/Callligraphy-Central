### Presentation Standards
*   **Design Alignment**: Layout rules are maintained inside `style.css`.
*   **Device Responsiveness**: Preserve grid rendering configurations (`insta-grid`, `insta-card`) to fit mobile/desktop breakpoints.
*   **Visual Assets**: Do not modify or delete branding media (`backg.png`, `calli_head001.png`, etc.).
