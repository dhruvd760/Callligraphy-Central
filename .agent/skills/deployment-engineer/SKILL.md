### Deployment Steps
*   **Environment Standard**: Configured to run on Apache 2.4 / PHP 8.x (XAMPP default layout).
*   **Permissions**: Ensure `uploads/` folder is writable by the web server daemon.
*   **Database Init**: Run `database.sql` to populate default table states.
